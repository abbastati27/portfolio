Coverly is a Streamlit-based application that automatically crafts professional, customized cover letters using your resume and a job description. It also calculates a candidate-job match score with a concise AI-generated explanation.


The app demonstrates end-to-end integration of document parsing, large language model prompts, and an interactive UI for instant cover letter generation.



Key Features & Highlights:

  • ✍️ Tailored Cover Letters: Extracts key details from the resume and job posting to generate a polished, role-specific letter.
  • 📊 Match Scoring: Provides a 0-100 candidate-job fit score along with a brief rationale.
  • 📂 Multi-format Input: Accepts resumes in PDF, DOCX, or TXT and automatically parses the text.
  • ⚡ Instant Download: One-click export of the final cover letter as a TXT file.
  • 🌐 Streamlit UI: Clean, interactive interface for a seamless user experience.
