CareerPathAI is an intelligent career counseling chatbot built using large language models (LLMs) that guides students in identifying suitable career paths based on their academic background, interests, and skill sets.


The bot uses contextual reasoning and real-world examples to suggest courses, specializations, and career opportunities.



Key Features:

  • 🧠 LLM-Powered Counseling: Natural language interaction using GPT-4 to provide personalized career guidance.
  • 🎓 Smart Recommendations: Recommends career options, educational paths, and in-demand job roles with justifications.
  • 🔍 Course & Job Fetching (Optional): Uses REST APIs to fetch relevant online courses or job listings.
  • 📊 Interactive UI: Built using Streamlit to allow users to input their details and instantly receive suggestions.


Skills: Python (Programming Language) · LangChain · Streamlit · REST APIs
